# Website Double B

Site web de prezentare pentru o firma specializata in instalatii termice,
climatizare si ventilatie.

## Tehnologii

- HTML
- CSS
- JavaScript

## Functionalitati

- ecran de incarcare;
- fundal animat cu particule;
- animatii declansate la derulare;
- slider pentru portofoliu si furnizori;
- sectiuni pentru servicii, portofoliu si contact.

## Rulare locala

Deschide fisierul `index.html` intr-un browser.

## Publicare cu GitHub Pages

Repository-ul poate fi publicat direct prin GitHub Pages deoarece
`index.html` se afla in radacina proiectului.
